## Installation

You need to have Python 3.11 or newer installed on your system.
If you don't have Python installed, we recommend installing [uv][].

There are several alternative options to install ngff-transformations:

<!--
1) Install the latest release of `ngff-transformations` from [PyPI][]:

```bash
pip install ngff-transformations
```
-->

1. Install the latest development version:

```bash
pip install git+https://github.com/BioImageTools/ngff-transformations.git@main
```
